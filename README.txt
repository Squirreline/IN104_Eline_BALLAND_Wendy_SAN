Instructions pour faire compiler le projet :
1) make
2) ./end_file
 
Si jamais la compilation ne fonctionne pas avec "make", merci d'essayer avec "make -B"

Note : pour le Pacman, nous avons décidé d'utiliser la fonction scanf qui permet de lire les caractères ou chaînes de caractères saisies au clavier. Cette fonction ne peut pas lire les flèches directionnelles.
